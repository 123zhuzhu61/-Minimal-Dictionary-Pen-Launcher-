# 极简词典笔桌面启动器 (Minimal Dictionary Pen Launcher)

## 📌 项目简介
这是一款专为词典笔 / 低配置安卓设备打造的极简桌面启动器，最低兼容 Android 5.0（理论上支持安卓4.4，未测试），零冗余设计，资源占用极低，可实现开机自启、双界面切换、网页快捷方式管理等核心功能，让老旧设备也能拥有清爽高效的使用体验。
## ✨ 核心功能
极简双界面：主屏（仅显示常用 APP）+ 全部应用界面，一键切换 + 时间显示
开机强制自启：设备重启后自动进入本桌面，不回退系统桌面
快捷方式管理：支持添加网页链接（可用指定浏览器打开）到主屏，
横屏适配：专为词典笔横屏设备优化布局
极致轻量：无多余特效，流畅运行于低配设备
## 🖱️ 使用方法
1. 添加应用到主屏
👉 在全部应用页面
长按任意应用图标 → 自动添加到主屏
2. 删除主屏应用图标
👉 在主屏页面
长按应用图标 → 确认后从主屏移除
3. 界面切换
点击左侧蓝色竖条按钮，一键切换「全部应用 / 返回主屏」
